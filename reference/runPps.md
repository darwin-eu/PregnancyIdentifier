# runPps

Runs PPS algorithm

## Usage

``` r
runPps(cdm, outputDir, uploadConceptSets = FALSE, logger, ...)
```

## Arguments

- cdm:

  (\`cdm_reference\`)

- outputDir:

  output directory

- uploadConceptSets:

  if concept sets should be uploaded

- logger:

  (\`logger\`) Logger object.

- ...:

  optional parameters

## Value

cdm object
