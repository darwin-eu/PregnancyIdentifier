# export

Exports the patient level results to summarised share-able csv-files.

## Usage

``` r
export(cdm, outputDir, exportDir)
```

## Arguments

- cdm:

  (\`cdm_reference\`) CDM-Reference

- outputDir:

  (\`character(1)\`) The path to the outputs generated by \`runHipps()\`

- exportDir:

  (\`character(1)\`) The path to the folder where the sharable outputs
  should be written to.

## Value

\`NULL\`
