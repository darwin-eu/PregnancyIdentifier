# insertPregnancyEpisodesTable

Inserts the identified pregnancy episodes table into a database.

## Usage

``` r
insertPregnancyEpisodesTable(cdm, path, tableName, overwrite = FALSE)
```

## Arguments

- cdm:

  (\`cdm_reference\`)

- path:

  (\`character(1)\`) Path to the generated rds-file generated by
  \`runHipps()\`.

- tableName:

  (\`character(1)\`) Name of the table

- overwrite:

  (\`logical(1)\`: \`TRUE\`) If the table should be overwritten or not.

## Value

(\`cdm_reference\`)
