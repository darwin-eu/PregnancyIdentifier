# runHip

Runs the HIP

## Usage

``` r
runHip(cdm, outputDir, logger, ...)
```

## Arguments

- cdm:

  (\`cdm_reference\`) A CDM-Reference object from CDMConnector.

- outputDir:

  (\`character(1)\`) Output directory to write output to.

- logger:

  (\`logger\`) Logger object.

- ...:

  Dev params

## Value

cdm object
