# runEsd

Runs ESD

## Usage

``` r
runEsd(HIPPS, cdm, outputDir, uploadConceptSets = FALSE, logger, ...)
```

## Arguments

- HIPPS:

  (\`data.frame\`)

- cdm:

  (\`cdm_reference\`)

- outputDir:

  (\`character(1)\`)

- uploadConceptSets:

  if concept sets should be uploaded

- logger:

  (\`logger\`) Logger object.

- ...:

  Extra (development) parameters

## Value

\`NULL\`
