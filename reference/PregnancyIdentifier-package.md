# PregnancyIdentifier: Identifies Pregnancy Episodes in the OMOP (Observational Medical Outcoms Partnership) CDM (Commen Data Model)

Identify pregnancy episodes in the OMOP CDM, based on the HIPPS
algorithm described in Smith et. all (2024)
[doi:10.1093/jamia/ocae195](https://doi.org/10.1093/jamia/ocae195) .
